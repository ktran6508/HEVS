# HEVS - Heterogeneous Embedded Vision System

A professional camera accessory that attaches to any camera with HDMI output, displays a life monitor on a phone browser with real-time overlays, controls the camera remotely, and enables NFC tap-to-share for wireless photo transfers. 

Build on Raspberry Pi 5 -> CM5 + STM32G4 + FreeRTOS

## System Architecture

### Universal Mode (HDMI input from any camera)

### Native Mode (MIPI CSI-2 direct sensor)

## Hardware
- Raspberry Pi 5 8GB - main development platform
- Raspberry Pi 5 CM5 - final production target
- STM32G4 NUCLEO-G474RE - real-time subsystem
- Sony A6000 - development camera
- Guermok USB3 HDMI capture card
- Samsung Pro Endurance 128GB SD card
